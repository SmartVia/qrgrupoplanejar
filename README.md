# Grupo Planejar Linktree

Página pronta para GitHub Pages.

## Como publicar

1. Suba todos estes arquivos para um repositório no GitHub.
2. No GitHub, entre em `Settings > Pages`.
3. Em `Build and deployment`, selecione `Deploy from a branch`.
4. Escolha a branch `main` e a pasta `/root`.
5. Salve e aguarde o link do GitHub Pages.

## Apresentação institucional

Coloque o PDF da apresentação neste caminho:

`assets/apresentacao/apresentacao-institucional.pdf`

O botão "Apresentação Institucional" já está configurado para esse arquivo.
